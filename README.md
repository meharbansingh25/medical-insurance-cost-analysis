# Medical Insurance Cost Analysis

## Tools Used
- Excel
- SQL
- Power BI

## Project Overview
This project analyzes medical insurance costs using Excel, SQL, and Power BI dashboards.

## KPIs
- Total Patients
- Total Charges
- Average Charges
- Average BMI
- Max Charges

## Business Insights
- Smokers have higher medical charges.
- Obese patients show higher insurance costs.
- Senior patients spend more on healthcare.
