# Medical Insurance Cost Analysis

---

## Tools Used
- Excel
- SQL
- Power BI

---

## Project Overview
This project analyzes medical insurance costs using Excel, SQL, and Power BI. The project focuses on healthcare cost analysis, risk analysis, KPI tracking, and dashboard visualization to identify important business insights.

---

## Project Workflow
- Data Cleaning
- Data Analysis using SQL
- Pivot Table Analysis in Excel
- KPI Calculation
- Dashboard Development
- Business Insights Generation

---

## KPIs
- Total Patients
- Total Charges
- Average BMI
- Average Charges
- Maximum Charges

---

## SQL Analysis Performed
- Average Charges by Smoker
- Region-wise Cost Analysis
- BMI Category Analysis
- Age Group Analysis
- Risk Category Analysis using CASE WHEN
- Ranking Analysis using Window Functions
- CTE-based Analysis

---

## Business Insights
- Smokers have significantly higher medical charges
- Obese patients show higher insurance costs
- Senior patients spend more on healthcare
- High BMI increases insurance expenses
- Southeast region contributes higher medical charges

---

## Dashboard Features
- Interactive Filters
- KPI Cards
- Healthcare Cost Analysis
- BMI Category Analysis
- Region-wise Analysis
- Customer Risk Analysis
